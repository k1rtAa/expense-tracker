## Описание программы
**Expense Tracker** — это графическое приложение (GUI) на Python для удобного учета личных финансов. 
Программа помогает пользователю контролировать свои траты, классифицировать их по категориям и анализировать расходы за определенные периоды.

**Ключевые возможности:**
- Добавление записей с указанием суммы, категории и даты.
- Автоматическое сохранение всех данных в файл формата JSON.
- Фильтрация расходов по категориям.
- Мгновенный расчет общей суммы трат по выбранному фильтру.
- Валидация ввода: проверка корректности суммы и формата даты.

## Требования для запуска
Для работы программы необходим установленный интерпретатор Python версии 3.8 или выше.
Дополнительные библиотеки не требуются (используются встроенные `tkinter`, `json` и `datetime`).

---

## Как запустить
1. Склонируйте репозиторий или скачайте архив с кодом.
2. Откройте терминал/командную строку в папке проекта.
3. Запустите приложение командой:
```bash
python main.py
```

---

## Как пользоваться
1. **Добавление расхода:** Введите сумму (число), выберите категорию из списка и укажите дату (по умолчанию стоит сегодняшняя). Нажмите "Добавить расход".
2. **Просмотр:** Все записи отображаются в таблице в нижней части окна.
3. **Фильтрация:** Выберите нужную категорию в выпадающем списке фильтра и нажмите "Применить фильтр". Таблица обновится, а в поле "Итого" отобразится сумма только по этим тратам.
4. **Сохранение:** Данные сохраняются автоматически в файл `expenses.json` при каждом добавлении.

---

## Примеры работы (Тест-кейсы)
- **Позитивный тест:** Ввод суммы `500.50`, категория `Транспорт`, дата `2026-04-26`. Результат: запись успешно добавлена в таблицу и файл.
- **Негативный тест (сумма):** Ввод текста `сто` или отрицательного числа `-50`. Результат: всплывающее окно с ошибкой "Сумма должна быть положительным числом!".
- **Негативный тест (дата):** Ввод даты в формате `26/04/2026`. Результат: ошибка "Неверный формат даты! Используйте ГГГГ-ММ-ДД".
